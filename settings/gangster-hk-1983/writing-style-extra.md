# 港综写作风格补充规范 (Gangster-HK-1983 Writing Style Extra)

> 本文件由 `setting pack` 注入。Generator 加载基础 `rules/writing-style-core.md`
> 之后再叠加本文件。仅适用于港综同人题材。

---

## 一、语言风格

### 粤语融入

对白适度使用粤语词汇，但要普通话读者能看懂。避免通篇粤语（除非是帮派场景需要地道气氛）。

- 常用：**咪郁**（别动）、**扑街**（混蛋）、**靓仔/靓女**、**睇场**（看场子）、**大佬**、**兄弟齐心**
- 粗口可带英式：ICAC、差馆、爱丽丝
- 绝不全粤语：读者看不懂 = 劝退

### 典型场景词汇

- 茶餐厅 / 冰室 / 大排档 / 鱼蛋粉 / 云吞面 / 菠萝包
- 时钟酒店 / 板间房 / 骑楼 / 唐楼 / 公屋 / 工厂大厦
- 的士 / 电车 / 山顶缆车 / 天星小轮
- 差馆 / 警署 / PTU / ICAC / 廉记（廉政公署）
- 社团 / 堂口 / 红棍 / 草鞋 / 白纸扇 / 坐馆

### 语气与口吻

- **冷笔，不抒情**。主角思考时用动词而非形容词。
- **市井气**。不要文青句式（❌"他的眼神如同夜空中最亮的星"）。
- **幽默要带讽刺**。港片式自嘲：说最荒诞的话用最正经的语气。

## 二、场景感

### 必写的细节类型

- **物价**：一碗面 HKD 5、月租 HKD 1200、月薪文员 2500
- **街道**：具体到"德辅道中 / 通菜街 / 东头村道 / 庙街夜市"
- **气味**：地下水沟、烧腊、中药、烟草、海腥
- **声音**：吊扇、收音机（徐小凤 / 许冠杰）、麻将、街市叫卖
- **人情**：茶水钱、欠条、交数（分账）、还人情

### 禁区

- 禁现代品牌。1983 没有 iPhone、没有星巴克、没有 ZARA。
- 禁穿越梗。主角可以用后世知识暗喜，但不能嘴上说"香港人未来二十年..."。
- 禁虚构历史人物称真名。用代指："霍官泰、李超人、邵老板、包船王"。

## 三、港综特有题材规范

### 1. 三七开日常质量

日常剧情占约三成。**万物皆为饵**——每一顿饭、每一次街头偶遇都是未来剧情的伏笔或钩子。不能写一章"主角吃早餐"却对主线零推进。

### 2. 精准时间线

对 1983-2000 真实香港事件严苛校准。谁在哪一年活着、哪一年死了、哪一年港元贬值、哪一年交易所改制——绝不搞错。时间线精准本身就是戏剧冲突的来源。

### 3. 原创与经典结合

经典港片 / 港剧场景可以参与（如廉署肃贪风暴、黑色星期六、联交所成立、中英联合声明），但主角必须**深度介入**——不走一遍流程，而是让原剧情因为他的存在走向更符合他利益的方向。

### 4. 创作自检 Checklist

Generator 写完每章要自问：

- 本章涉及的历史事件年份对不对？
- 主角做这件事：有困难吗？符合人设吗？能闭环吗？
- 有没有描写那个年代的食物、衣服、街道、气味？
- 粤语融入是否过度？普通话读者能否读懂？
- 对照 `era.md` 和 `characters.yaml` 检查过了吗？

### 5. 拒绝闭门造车

学习优秀港综同人的四方面经验：**人物弧光**（不偏离原型但挖掘新层次）、**剧情切入点**（利用"时间缝隙"或"未解之谜"）、**世界观深化**（让金手指合理融入而非突兀）、**情感互动**（即使不写 CP，人物间张力必不可少）。
