# 末世虫潮 · DNA 卡

## 一句话定位
末世虫潮：主角以虫族寄生+修复金手指碾压神话体系，守护日常为核心爽感的数值膨胀型末日文。

## 戏剧结构
- **整书基调**：末日求生外壳包裹"守护日常"的情感内核，战斗暴力与家庭温情强对比，穿插性别反转、伪娘误会等网络喜剧元素。
- **典型的"5 章情节单元"长什么样（起-承-转-合的样板）**：
  - **起**（1章）：日常基线或新副本开启，古月与姐姐/苏菲/纽克斯的温情互动，或空间跳跃至新地图；
  - **承**（1-2章）：威胁浮现（虫族/幻想体/神道组织），信息碎片化释放，古月评估数值→制定策略；
  - **转**（1章）：核心反转——金手指升级（修复逆天改命）、虫族底牌亮相（"比人多"）、规则级秒杀（因果反转），或盟友天降破局（兰染）；
  - **合**（1-2章）：战利品分配/情感确认（拥抱、拜师、吃醋），新建悬念（新演化物、新威胁预告）。
- **信息释放节奏（作者如何管理读者的好奇心）**：
  - 世界观补丁采用"后置补完"（古老密林时间流速、幻想体产生机制在战斗中突然抛出）；
  - 关键信息错位：读者先于主角（古月已知纽克斯身上有生命之瓶，沈明不知）或主角先于读者（古月直接跳跃结论，不展示推理）；
  - "故意不给"：裸体女人身份、虫神本体状态、神道组织演化物总数等数字黑洞悬置。
- **爽点频率（每几章一个小爽点 / 大爽点）**：
  - 小爽点：每1-2章（战利品获取、同伴震惊、吐槽喜剧）；
  - 大爽点：每4-5章（虫族底牌反转、规则级秒杀、绝境翻盘）。

## 写作风格
- **叙述视角 + POV 策略**：
  - 名义单POV（古月），实为"古月锚定+频繁跳切"的伪全知：战斗间隙插入宙斯/奥丁/托尔/曹吉等副视角，学习段落插入梅林心理崩溃；
  - 关键场景嵌入第一人称回忆（沈明希腊历险）或神话科普旁白（托尔地位变迁注释）。
- **句式 / 节奏 / 对白密度（定量描述）**：
  - 战斗场景：短句占比80%，平均每段≤30字，段落长度≤3行，感叹号密度每章15-20个；
  - 日常/解释场景：长句堆叠，单段超200字无分段，时间词（"一个星期又一个星期"）拖拽节奏；
  - 对白占比：战斗场景15%-20%，日常场景40%-50%，情报交换场景近乎纯对白驱动。
- **特色修辞 / 高频词 / 作者口味**：
  - 网络口语直嵌："死道友不死贫道""华丽的晕倒""神马～～～""搞基""五姑娘"；
  - 游戏术语侵入："下副本的队伍搭配""BUG""YY小说""登哥"；
  - 数字偏执：战斗力"五亿""百亿""一兆伏特"量化一切，时间流速"外面一天这里好几年"；
  - 拟声词+动作标注："咔嚓一声""泪奔""虎躯一震"；
  - 括号式科普插入："滑头鬼，源于岛国民间传说中的客人神……"。
- **章末钩子的三种主力款式**：
  1. **威胁升级型**："地面开始震动，无数百米长的甲虫出现了""翻过一座山，两人身体一僵……"（省略号截断）；
  2. **计谋萌芽型**："古月心中一动，计上心来"（具体计谋延迟至下章首段兑现）；
  3. **身份/情感悬疑型**："那个手，不觉得熟悉么？""哥向来爷们，要不比比？"——性别/身份反转的荒诞悬置。

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - 底层：收集/吞噬升级（七星神炎、演化物）；
  - 表层：守护姐姐→守护苏菲/纽克斯/希望城→"为了这样的和谐日常，绝对不能让虫族得逞"；
  - 关键词：效率最大化、无道德犹豫、控场式微笑、精神力扫描先行、资源分配者。
- **主角的金手指是什么 + 作者如何避免主角"开挂无趣"**：
  - 金手指矩阵：修复能力（逆天改命）、虫族寄生网络（规模效应）、空间跳跃（机动）、精神力化身（一心两用）、葬神戟（器灵化/因果级）；
  - 防无趣机制：①能力"使用中升级"（肉丝同化反哺本体）；②限制条件（修复消耗、虫族控制需"三个星期"意志对抗）；③被动场景插入（被托尔突袭、被双演化物压制）；④依赖盟友破局（兰染天降、夏舒失控龙卷风误伤）。
- **典型反派形象 + 反派的"真实威胁值"如何维持**：
  - 形象：神话体系降格（奥丁窥视、托尔急躁、月读本体极弱）、神道组织演化物持有者（田岛"杀不死"）、规则级存在（背棺者棺材里的手）；
  - 威胁维持：①信息差（古月不知乌鸦有第二只）；②数量压制（神道组织演化物总数模糊）；③"未死即召唤"的僵局（田岛）；④死亡预告悬置（"等我本体苏醒"）。
- **本书价值观锚点**：读者被什么打动？
  - **爽感**（数值碾压、信息差打脸）> **守护日常的血性**（"为了这样的和谐日常"）> **荒诞幽默**（沈明性别反转、伪娘误会）> **成长**（古月从独狼到召集集训、全球召唤）。

## 核心设定（**供'禁止直接抄袭'清单用**）
- **世界观公理（3-5 条）**：
  1. 虫潮从2021年冬天开始，幻想体/神话生物随世界树孕育加速涌现（五十年前力量增长加速）；
  2. 古老密林等区域时间流速与外界不同（外面一天=里面数年），催生高战力生物密集区；
  3. 新人类通过"进化"升级，存在心脏崩溃死亡率，可被修复能力打破；
  4. 东西方幻想体体系完全不同（西方神谱vs东方修仙元婴）；
  5. 演化物为末世核心争夺资源，能力命名依赖成语/俗语改造（"天网恢恢""苦海无边""真实的谎言"）。
- **主角专属设定（金手指来源、身份秘密、真实立场）**：
  - 金手指来源："活了无数岁月的老妖怪"精神力百不存一，虫族底蕴来自古老密林"几千亿年"时间差；
  - 身份秘密：身体器官"老早就没了"，无心脏、无器官，免疫剥离类能力；
  - 真实立场：表面收集者，实际"不一定要得到演化物，只要阻止神道组织得到"的破坏者。
- **专属角色 / 组织 / 地名（列出前 5 个最具辨识度的）**：
  1. 希望城（古月守护的日常基线城市）；
  2. 不灭城（与希望城对标的另一人类据点）；
  3. 古老密林（时间流速差异区，虫族繁殖基地）；
  4. 罗兰城（梅林魔法教学地，亚特兰蒂斯阵法节点）；
  5. 神道组织（演化物收集者，反派主线组织）。
- **专属物品 / 技能 / 概念名词（列出前 5 个）**：
  1. 葬神戟（器灵化武器，因果级攻击"因果反转"）；
  2. 修复能力（古月核心金手指，可修复心脏崩溃）；
  3. 虫族寄生肉丝（特洛伊木马式渗透，同化反哺本体）；
  4. H粒子炮（科技侧碾压武器）；
  5. 演化物（"正反""天网恢恢""剥离"等规则级道具）。

## 可借鉴的操作手法
**（这部分是融合新世界观时可直接借鉴的抽象技法，不含任何原著专属名词）**

- **情节层**（3-5 条"写起承转合时的句法"）：
  1. **"日常-爆破-日常"节拍器**：每单元以温情日常起，以威胁爆破转，以新建日常合，形成情绪过山车；
  2. **"数值锚定→视觉压迫→机制揭示"三阶BOSS引入**：先报数字建立预期，再铺陈体型/场景压迫，最后揭示特殊机制制造障碍；
  3. **"能力喊名+反复操作"爽点固化**：危机→诊断→指令→动作→能力释放→"崩溃一次/修复一次"的重复节奏→结果确认；
  4. **"信息差碾压"替代直接战斗**：主角已知而反派不知的关键情报（位置/身份/能力），转化为同伴震惊或反派崩溃的情绪爽点；
  5. **"死亡预告悬置"替代即时击杀**：反派败退时留下"等我本体苏醒"等未兑现威胁，维持长期张力。

- **风格层**（3-5 条"具体到可套用的语言招式"）：
  1. **独立成段冲击句**：核心反转信息单独成段，加多感叹号（"沈明变成女人了！！！"）；
  2. **"微笑+暴行"反差修辞**：主角以"笑眯眯"等温和表情修饰暴力行为，消解仪式感；
  3. **科普插入打断**：战斗/对话中突然插入括号式百科注释，制造"伪纪录片"质感；
  4. **时间副词机械加速**："下一刻""瞬间""骤然"每章12-15次，强制推进动作节奏；
  5. **数字量化一切**：将抽象战力、时间、概率全部转化为具体数字，建立游戏化认知框架。

- **人物层**（3-5 条"人物关系 / 介绍 / 发展的招式"）：
  1. **"标签+单次场景"极速定性**：新角色出场3段内完成能力标签+外貌锚点+即死风险/喜剧功能；
  2. **"失踪→重伤→自述"三阶切入**：重要配角先以"被提及"状态悬置，再以身体登场制造冲击，最后长篇自述完成定性；
  3. **"吃醋组"喜剧调剂**：核心关系网外围配置2-3个嫉妒/误会型角色（纽克斯/波妮），在战斗间隙制造情感张力；
  4. **"主动追随"魅力认证**：强者角色以"拜师""加入小队"等主动请求，替代主角招募，强化主角吸引力；
  5. **"工具人退化"功能管理**：前期有自尊的导师型角色（梅林）被主角打击至麻木，彻底工具人化，避免关系复杂化。

## 主角主动度量化（汇总自 7 个窗口的『主角主动度』维度）

- **全书主动场景占比**：~62%（34/55场为主动）

- **主动 vs 被动的章节分布特征**：
  - **卷开头**：主动度极高（空间跳跃夺宝、制定寄生策略、全球召唤），古月以"评估→决策→执行"循环快速建立控制；
  - **卷中段/转折点前**：被动陡增（被托尔突袭、被双演化物压制、被月读百鬼夜行突袭），但被动场景平均持续≤1章即转为主动反制；
  - **卷结尾/铺垫下一卷**：主动与被动交织，古月"主动选择被动"的控场行为增多（阻止夏舒出手旁观、故意惊吓同伴），同时插入盟友自主线（曹吉独探岩浆）分散主角负荷。
