# 地球OL · DNA 卡

## 一句话定位
系统流+神话无限副本，主角以规则套利与信息差碾压，收割NPC与玩家建立神朝。

---

## 戏剧结构
- **整书基调**：游戏化生存的冷峻功利主义，穿插日式中二战斗美学与中式网文权力扩张快感，情绪曲线呈"压抑-爆发-收割-转场"的锯齿形。
- **典型的"5 章情节单元"长什么样（起-承-转-合的样板）**：
  - 起（1章）：新地图/新敌人入场，主角接收任务或遭遇突发危机，嵌入世界观碎片（如"斯巴达-亚马逊地理框架""十年备战收束"）
  - 承（2章）：试探性交手或团队分工，敌方底牌逐层剥开，主角藏招不揭（如十二战士车轮战、三战场并行分裂）
  - 转（1章中段）：金手指针对性亮相或规则突变，战局断崖式逆转（如真龙之眼破幻术、未知法则降维打击、职业变异）
  - 合（1-2章）：收割战利品/收编势力/能力验证，章末嵌入新悬念钩子（如"两张卡片"属性展示、小型空间秘密暴露）
- **信息释放节奏（作者如何管理读者的好奇心）**：
  - 效果先行、原理后置（真龙之眼、混沌之体、虚化机制均先展示效果再解释）
  - PS作者插话制造信息错位（"关雨攻击破不了希波吕忒防"读者先知）
  - 故意withholding：托妮死亡经过、宫南白禁闭原因、阿瑞斯实力暴涨原因等核心背景长期留白
  - 战斗中释放机制说明（服部半藏"贼曹"官位效果在交锋时详解）
- **爽点频率（每几章一个小爽点 / 大爽点）**：
  - 小爽点：每1-2章（秒杀单个敌人、技能首秀、装备入手）
  - 大爽点：每4-5章（全族收编、职业变异、跨级反杀、势力扩张）
  - 超级爽点：每15-20章（灭界之战启动、神级博弈、长期伏笔兑现）

## 写作风格
- **叙述视角 + POV 策略**：
  - 主体为严格单POV（关雨），但存在三类越轨：①PS作者插话打破第四墙 ②"伪全知"漏洞补充设定 ③敌方视角/外部视角的临时切出（陈道临跟踪、漆原摩智心理、千目蝶视角）
  - 群战场景采用"画面回到X分钟前"的显性跳转，实为伪多POV
- **句式 / 节奏 / 对白密度（定量描述）**：
  - 战斗场景：短句碎切占70%，平均每句≤15字，感叹号密度高（"发射！""不好！""怎么可能！"）
  - 日常/设定场景：长句堆叠，连续200字无分段，定语从句密集（女性身体描写连续4个定语修饰）
  - 对白占比：战斗场景25%-30%，日常场景35%-40%，文戏场景可达50%以上
  - "动作-解说-反应"三段式循环，每轮攻防后必接心理活动
- **特色修辞 / 高频词 / 作者口味**：
  - 游戏术语强制嵌入叙事："没破防""血掉了1%""状态窗口""副本主线任务"
  - 神话/佛教/日式技能名混用："六道轮回经""香取神道流·霞之剃刀""射杀他，神枪！"
  - 数字量化偏好："三千米""一百米外""二十四枚""实力提升120%""一亿倍"
  - 女性身体"数据化审美"："木瓜蛇腰""加分无数""有力又修长的大腿"
  - 网络用语："奶奶个熊""破鞋黑木耳""跑跑君""猥琐"
  - 引号强调癖：大量用引号标注特殊含义或反讽（"'艰难'的闯关"）
- **章末钩子的三种主力款式**：
  1. **系统公告/属性面板型**："任务面板弹出，竟然诡异的改变了副本主线任务""恭喜玩家ID0成功传承职业……命名为：【混沌死灵师】（唯一）"
  2. **外部视角震惊型**：陈道临"阴兵借道"误判、温哥伦"立即有所发现"、御帝抬头看"升腾而起的白光"
  3. **身份/规则颠覆型**："老头变俊美男子""此ID玩家姓名错误，无法传送""未知法则"亮相

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - 表层动机：复活托妮（贯穿全书的主线任务）
  - 深层动机：从"谨慎风险计算"到"帝王式权力扩张"的风格演进，饥饿感/生存本能驱动
  - 性格关键词：功利、藏招、规则套利、被动承受→主动转化、道德焦虑消解（希波吕忒按手于胸的台词功能）
- **主角的金手指是什么 + 作者如何避免主角"开挂无趣"**：
  - 金手指体系：①系统任务/奖励机制 ②装备叠加（尼密阿之戎装+战神腰带+界王锤）③职业变异（混沌死灵师）④血统/法则（虚化、混沌之体、真龙之眼）⑤官位体系（贼曹等神朝职位）
  - 防无趣策略：①信息差制造虚假危机（读者知主角破不了防，主角不知）②前置布局代价（表皮战术需预先分离本体，变异过程七天无呼吸）③更高层级力量接管时的被动性（哈迪斯试验品、系统强制灌注）④饥饿感/时间压力等生理限制
- **典型反派形象 + 反派的"真实威胁值"如何维持**：
  - 形象一："努力型反派"（漆原摩智、黄金三角）——底牌尽出、组合技华丽，被规则级差距碾压，威胁来自"未知道具"（最后的晚餐）
  - 形象二："降维打击者"（阿蒙神）——未知法则、信息黑箱，主角方从碾压转入溃退，靠第三方打断（梵天）而非主角解决
  - 形象三："神级博弈者"（哈迪斯、宫南白）——非敌对而是利用/考验关系，威胁来自"兴趣转移"和"试验品"身份
  - 威胁维持：①延迟释放底牌（阿蒙神未知法则第4章才亮相）②假名/假ID机制逃脱③第三方救援打断（泰芙努特救荷鲁斯）
- **本书价值观锚点**：读者被什么打动？
  - 核心：爽感（防御碾压的安全感、全族收编的权力幻想、数值膨胀的量化快感）
  - 次级：生存智慧（规则套利、信息差计算、前置布局）> 血性（洪成XP技能情绪爆发）> 荒诞幽默（库克诺斯"事不过三"、元小说自我指涉）> 哲思（哈迪斯"众神寿命太过悠长"的bored神格）

## 核心设定（**供'禁止直接抄袭'清单用**）
- **世界观公理（3-5 条）**：
  1. "地球OL"是真实存在的游戏系统，玩家死亡可复活（有代价），NPC可被"卡片化"收编
  2. 多元神话体系并存且可交互（希腊冥界、埃及神系、修仙六大门派、日本神道），存在"投影/真身"区分
  3. 存在"主神空间""副本空间""小型空间"等多层空间结构，时间流速可差异
  4. "XP技能"机制：情绪值达标触发，10级后领悟，每日限次限时
  5. 神朝/帝国体系可赋予"官位"克制能力（如"贼曹"克盗贼/盗取法则者）
- **主角专属设定（金手指来源、身份秘密、真实立场）**：
  - 金手指来源：系统玩家身份+哈迪斯外力干预职业变异+宫南白传承《六道轮回经》
  - 身份秘密：ID0（最早/特殊玩家编号）、混沌死灵师（唯一职业）、表皮战术（液体本体可分离）
  - 真实立场：表面任务驱动（复活托妮），实际逐步滑向"不朽神朝"帝王，与NPC/玩家的权力关系模糊化
- **专属角色 / 组织 / 地名（列出前 5 个最具辨识度的）**：
  1. 不朽神朝（关雨建立的跨世界帝国，含"贼曹"等官职体系）
  2. 宫南白/败地仙尊（修仙线传承者，"猥琐老头→俊美男子"反转）
  3. 哈迪斯（冥王，职业变异的"试验品"推手）
  4. 希波吕忒/亚马逊女王（首个全族收编的"后宫+部下"双重身份NPC）
  5. 安倍晴明（军师型角色，负责布局与调度）
- **专属物品 / 技能 / 概念名词（列出前 5 个）**：
  1. 尼密阿之戎装（S级防御装备，"没破防"核心爽点来源）
  2. 战神腰带/折射腰带（希波吕忒奖励，技能"折射"）
  3. 六道轮回经（修仙功法，与冥界机制相似，含"十八大地狱"等）
  4. 混沌死灵师（唯一职业，无品阶，四技能体系）
  5. 灵核弹/三核爆弹/三核龙皇剑（科技+修炼混合的量产型大规模杀伤武器）

## 可借鉴的操作手法
**（这部分是融合新世界观时可直接借鉴的抽象技法，不含任何原著专属名词）**

- **情节层**：
  1. "藏招-破局-秒杀"三段式战斗：先让敌方完整展示底牌，再揭露针对性克制手段，以原始暴力（穿胸、捏爆）而非华丽特效终结，制造"降维打击"的反差爽感
  2. "被动获益"的借势模式：让更高层级力量的对抗溢出（神级vs系统、法则vs法则）惠及主角，主角作为容器/见证者承受变异，以短期主动性让渡换取超额收益
  3. "先扬后抑再扬"的波浪结构：连续小爽点后突然中断，以敌方隐藏底牌制造危机，再通过前置布局（表皮战术、假死）二次反转
  4. "全族/全员收编"的权力扩张节点：将击败的敌人转化为可调用资源（卡片化、封印、官职授予），爽感从"击杀"升级为"所有权"
  5. "第三方打断"的悬念冻结：在即将决出胜负时引入更强仲裁者，将生死战转化为政治博弈，延长张力周期

- **风格层**：
  1. "效果先行、原理后置"的信息释放：先展示技能/能力的视觉奇观，延迟数章再解释机制，期间制造"如何破解"的悬疑
  2. "PS作者插话"的信息错位：以元叙事评论打破第四墙，向读者泄露主角不知的关键信息（"其实破不了防"），制造讽刺张力
  3. "数据化审美"的身体/实力描写：用数字、等级、百分比替代传统形容词，将感性体验转化为可比较的量化指标
  4. "敌方台词功能化"的对话设计：反派对白缩减为"惊呼-误判-求饶"三件套，作为主角实力的反应器，不赋予独立人格表达
  5. "系统公告型"章末钩子：以规则面板、属性弹窗、任务变更等机制性文本替代传统悬念描写，形成"游戏内通知"的硬质截断感

- **人物层**：
  1. "功能先行，背景后补"的新角色引入：先给角色一个可立即使用的功能标签（破阵、送离、官位克制），再逐步补完个人动机，降低认知负荷
  2. "四步女王切入法"：先声夺人（声势/击杀）→ 肉体诱惑/身份反差 → 实力验证 → 情感绑定（将权力移交转化为情感契约），快速完成"强者+爱人+部下"三重叠加
  3. "抑扬式弱者逆袭"：先让角色展示缺陷（口吃、怯战、退后），再延迟揭示隐藏能力，以"撒谎诱敌"完成反杀
  4. "标签回收"的工具人管理：给次要角色一个易记绰号/特征（"旋风姑娘"），在数章后将其转化为具体功能，制造"早该想到"的伏笔快感
  5. "外部视角镜像"：引入与主角无直接关系的观察者角色，通过其误判（"阴兵借道"）和震惊反刍主角威势，实现"社会传播"层面的爽感扩散

- **主动 vs 被动的章节分布特征**：
  - **卷开头**：主动度高（布局、决策、出征），如ch123提出灭界之战、ch054夜谈布局
  - **核心战斗段**：主动与被动交替，"藏招-破局"为主动，"中招-承受变异/伤害"为被动，形成"主动试探→被动承压→主动反杀"的波浪

  - **特殊被动模式**："借势型被动"——主角主动选择让渡身体控制权（接受变异、分离表皮），以短期被动换取长期主动，是本书区别于传统无敌流的关键设计
