# 末日生存方案供应商 · DNA 卡

## 一句话定位
现代人用营销话术与信息差在古代末日世界降维打击。

## 戏剧结构
- **整书基调**：荒诞喜剧 + 智力碾压 + 热血扩张。主角用现代思维（营销、诡辩、信息战）在原始/玄幻末日背景下不断“忽悠”原住民，形成强烈的认知错位笑点与爽感。战斗是点缀，核心爽感来自“一本正经胡说八道”的智力征服。

- **典型的“5 章情节单元”长什么样**：
  - **起**（第 1 章）：用日常或轻松场景建立当前状态，常以主角的某个“现代操作”（广告、跟踪、赴约）为起点，节奏舒缓，埋下冲突引子。
  - **承**（第 2 章）：视角切换或信息堆积，让读者看到“另一拨人在干什么”，与主角线形成信息错位，制造“他们即将撞上铁板”的悬念。
  - **转**（第 3 章）：**核心转折点**。冲突从暗转明，主角用非传统方式（身份揭露、诡辩洗脑、攻心瓦解、暴力羞辱）瞬间扭转局势，读者预期被打破，爽感集中释放。
  - **合**（第 4-5 章）：战后收束或信息交换。主角完成“身份坐实/资源获取/团队整合”，同时通过对话释放新的世界观信息或埋下更大伏笔，为下一单元做铺垫。

- **信息释放节奏**：
  - **先展示结果，后解释原因**：让读者先看到主角“守株待兔”或“卡点行动”的结果，制造惊讶，再通过内心活动或第三方复盘揭示其精密算计，形成“原来如此”的智力快感。
  - **“读者先于主角知道”的错位**：频繁使用。读者通过多视角切换，先知道有人在跟踪/潜入/心怀鬼胎，而主角看似毫无察觉，制造窥视感、喜剧张力或紧张感。
  - **通过对话释放人设与设定**：主角的“绿帽侠”人设、世界观规则、金手指机制，几乎都通过角色对话“不经意”透露，而非旁白硬性介绍。
  - **故意不给的信息**：主角的真实实力、最终目的、金手指的具体原理，始终守口如瓶。只用“手法”、“妓女之友”等戏谑回答转移探究，维持根本悬念。

- **爽点频率**：
  - **每 2-3 章一个小爽点**：身份揭晓（强敌变迷弟）、诡辩洗脑（学生喊口号）、广告效果爆炸（全城疯抢）。
  - **每 5 章一个大爽点**：界主降临秒杀、当众暴打羞辱反派、一拳 KO 顽固分子、师徒配合颠倒黑白。

## 写作风格
- **叙述视角 + POV 策略**：
  - 以**主角苏烬的第三人称有限视角**为主，读者所知基本与他同步。
  - **频繁但短促地切入配角/对手视角**（曲沐棠、赤祀、叶长风、保安老赵），用于提供外部观察、制造信息错位、或侧面烘托主角的深不可测。每次切入通常只持续几段，完成“定性”或“反应”功能后迅速切回。

- **句式 / 节奏 / 对白密度**：
  - **对白占比极高（超过 60%）**，是推动情节、塑造人物、释放信息的核心手段。
  - **句式长短结合，节奏感强**：战斗/冲突场景多用短句、断句、单字成段（“咚！咚！咚！”“死！死！死！”），营造冲击力；叙述/心理活动多用长句，放缓节奏。
  - **口语化强，充满市井粗口与现代网络用语**（“急头白脸”、“撒逼”、“臭婊子”、“家人们”），与玄幻/古代背景形成刻意割裂。

- **特色修辞 / 高频词 / 作者口味**：
  - **现代概念解构原始场景**：用“广告”、“弹幕”、“合影福利”、“商务场所”、“游戏体验”等现代词汇描述异世界行为，是核心笑点来源。
  - **游戏化数值表现**：战斗时直接出现“暴击-999”、“血条掉了三分之一”、“S 级评价”等 RPG 逻辑。
  - **夸张的群体反应**：频繁使用“全场死寂”、“倒吸凉气”、“表情像被雷劈过”等描写，侧面烘托主角言行的惊世骇俗。
  - **一本正经的诡辩**：用看似有逻辑的类比（如谷道与耳道的辩证）包装歪理，形成“流氓教师”式语言风格。

- **章末钩子的三种主力款式**：
  1. **未完对白 / 信息截断**：对话说到关键处戛然而止，或切到窃听者的反应（如“曲沐棠：？？”），制造即时悬念。
  2. **威胁升级 + 行动预告**：明确预告“陆家肯定回去叫人了”、“直接冒充魔道杀人行凶”，让读者对下一波冲突产生强烈期待。
  3. **资源奖励 + 身份确认**：主角获得新权限、新后盾或新目标（“一个完整的星球给你做后盾”），暗示下一阶段剧情方向。

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - **核心动机**：完成任务（确保目标存活/人心所向），本质是**生存与扩张**。具体表现为不断“解决问题，稳固发展”，积累资源与信用。
  - **性格关键词**：圆滑、无耻、冷静、掌控欲强。能瞬间切换嚣张与委屈两种状态，毫无心理负担地“忽悠”任何人。

- **主角的金手指是什么 + 作者如何避免主角“开挂无趣”**：
  - **金手指**：现代人的信息差与营销思维 + 超强身体素质（神经反应、天生神力）+ 薪火印记带来的特殊权限（请神、系统后门）。
  - **避免无趣的方式**：
    - 金手指的**具体原理始终不解释**，保持神秘感。
    - 胜利方式**以智力碾压为主，武力碾压仅为收尾**（先智取后力敌）。
    - 频繁用**伏笔回收**（如“没有战纹”的设定在战斗后才揭示其反差）来强化爽感，而非平铺直叙。

- **典型反派形象 + 反派的“真实威胁值”如何维持**：
  - **典型反派**：愚忠的绊脚石（严台）、被腐化的叛徒（袁夜生）、仗势欺人的炮灰（陆无生）、有理想但手段幼稚的反对派（赤祀）。
  - **威胁值维持方式**：反派通常**出场即被定性，但不会被立刻彻底消灭**。他们要么逃跑（袁夜生“转身就走”），要么留下狠话（秦长老“暂且记下”），要么其背后势力被预告将报复（陆家），将冲突从“一次性解决”转为“长期威胁”。

- **本书价值观锚点**：读者被什么打动？
  - **智力优越感**（核心）：看着主角用现代思维把原住民耍得团团转，产生强烈的智商碾压快感。
  - **荒诞幽默**：现代概念与原始背景的混搭，师徒配合“颠倒黑白”的闹剧。
  - **血性与爽感**：当众暴打羞辱反派、一拳 KO 的绝对武力释放。
  - **生存智慧**：主角在多方势力间游刃有余的圆滑与算计。

## 核心设定（**供“禁止直接抄袭”清单用**）
- **世界观公理**：
  1. 末日是真实存在的，且伴随植物疯长、虫潮等具体灾难。
  2. 存在一个名为“归衡台”或类似机制的上层组织/公司，可以连通不同世界，但因灾难传染风险，各世界被孤立。
  3. “薪火”是一种可传承的身份/权限系统，包容万道，能转化无限，但个人禀赋不同导致印记体现不同。
  4. 存在“战纹”体系，是原住民的核心力量来源，主角没有战纹但能通过其他方式（天生神力、润滑油分泌）与之对抗。
  5. 游戏系统（如天域游戏）是真实存在的，其中包含 AI 女友、单人生活副本、机甲对战等具体玩法。

- **主角专属设定**：
  - **金手指来源**：穿越者身份带来的现代知识 + 薪火印记 + 天材地宝速成能力（但只能转化为恢复力，增长需搏命）。
  - **身份秘密**：真实身份是“末日生存方案供应商”，但在不同世界使用不同马甲（张世豪、天域豪哥、孙啸川、合欢宗弟子）。
  - **真实立场**：表面是参与者，实则是利用一切资源为末日做准备，最终目的隐藏极深。

- **专属角色 / 组织 / 地名**：
  1. **苏烬**（主角，多马甲）
  2. **沈御风**（早期核心保护对象，预知者）
  3. **陆明轩**（五纹高手，被电影忽悠入队）
  4. **无极 / 袁夜生**（师徒，薪火传承的腐化案例）
  5. **赤祀**（有理想的反对派，被一拳 KO 收服）

- **专属物品 / 技能 / 概念名词**：
  1. **薪火印记**（主角的权限与能力来源）
  2. **归衡台**（跨世界连通机制）
  3. **战纹体系**（原住民力量核心）
  4. **天域游戏**（含 AI 女友、机甲对战等）
  5. **请神**（召唤高位存在附身的技能）

## 可借鉴的操作手法
- **情节层**：
  1. **“先结果后原因”的复盘式爽点**：让第三方角色（如张 sir）向蒙在鼓里的人复盘主角的全盘算计，读者获得“原来如此”的智力快感。
  2. **“螳螂捕蝉，黄雀在后”的多层跟踪**：让配角跟踪主角，另一个配角再跟踪他们，制造窥视感与喜剧张力。
  3. **“一本正经胡说八道”的诡辩洗脑**：用看似有逻辑的歪理邪说（如将羞耻感定义为恐惧）解构对手的道德立场，完成精神碾压。
  4. **“先智取后力敌”的组合拳收服**：先用攻心计瓦解敌方团体，再用绝对武力收服敌方首领，完成双重征服。

- **风格层**：
  1. **现代概念解构原始场景**：用“广告”、“弹幕”、“商务场所”、“游戏体验”等词汇描述异世界行为，制造反差幽默。
  2. **游戏化数值直接嵌入叙述**：战斗时直接写“暴击-999”、“血条掉了三分之一”，将抽象战斗转化为 RPG 逻辑。
  3. **夸张的群体反应侧面烘托**：频繁用“全场死寂”、“倒吸凉气”等描写，代替直接描写主角有多强。
  4. **对白风格刻意割裂**：主角团用现代粗口/网络用语，原住民使用传统玄幻/古代用语，形成喜剧反差。

- **人物层**：
  1. **通过“态度反转”快速定性新角色**：让一个喊打喊杀的敌人，在得知主角身份后瞬间变为圆滑讨好的“兄弟”，高效刻画其重利、懂变通的性格。
  2. **用“档案式口述”引入工具人**：通过主角布置任务时的口述，直接交代新角色的身份、弱点、在剧情中的工具性作用，毫无废话。
  3. **通过主角的压迫感塑造配角**：让心怀鬼胎的配角在主角看似随意实则精准的试探下紧张、心虚、恐惧，从而塑造其如履薄冰的形象。
  4. **为旧角色增加“观众视角”功能**：让已有人物在关键时刻作为“死斗见证人”出场，通过其反应（零食袋滑落）侧面烘托战斗结果的震撼。
