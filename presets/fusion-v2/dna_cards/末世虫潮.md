# 末世虫潮 · DNA 卡

## 一句话定位
末世强者带系统养队友，用信息差和智力碾压一切敌。

## 戏剧结构
- **整书基调**：快节奏末世爽文，表面残酷生存，内核轻松诙谐。战斗密集但不过度压抑，随时插入吐槽和冷幽默调节气氛。主角始终掌握主动权，危机是成长的催化剂而非真正的绝望。

- **典型的“5章情节单元”长什么样**：
  - **起（第1章）**：确立一个明确任务或冲突（追杀目标/收徒/寻宝/惩罚队员），通常以对话或主角决策开场。
  - **承（第2-3章）**：执行任务中遭遇阻碍或新变数（敌人太强/情报不足/队员失误），节奏从轻松转向紧张。
  - **转（第4章）**：核心转折点——主角用智力或金手指破局（赌赢林玄获得情报/杀死曹吉助其觉醒/借正义武器破解幻象），爽点释放。
  - **合（第5章）**：战果收割+新悬念埋设。胜利后立刻发现新目标或新威胁（奇异果实/连河果登场/冥海降临），钩子形成。

- **信息释放节奏**：
  - **分阶段释放金手指**：主角能力不是一次性交代完，而是通过战斗逐步解锁（如葬神戟属性面板上大量“？？？？”）。
  - **用对话交代设定**：世界观、升级体系、敌人情报几乎全通过角色对话释放，避免旁白说明。
  - **制造信息错位**：频繁使用“读者先于主角知道”的手法（如连河果的内心独白、企鹅族灭门事件），制造戏剧张力和期待感。
  - **扣留关键信息**：主角的真实想法、敌人的完整动机、道具的全部功能常被故意保留，维持神秘感。

- **爽点频率**：
  - **小爽点（每1-2章）**：队员实力暴涨、主角轻松戏耍对手、新武器属性展示。
  - **大爽点（每5章单元结尾）**：收服强敌、身份揭晓带来的震撼、用智力破局碾压。

## 写作风格
- **叙述视角 + POV策略**：
  - 主视角严格跟随主角古月（有限第三人称），但会频繁、短暂地切入其他角色的内心活动（如连河果的“春风得意”、胡燕的“渴望力量”），形成“类全知”的灵活视角。
  - 偶尔切换至完全独立于主角的副线场景（如北极企鹅族灭门），用于埋设跨章节的宏大伏笔。

- **句式 / 节奏 / 对白密度**：
  - **句式**：以短句和中等长度句子为主（15-25字），战斗时多用连续动作短句（“他身体一荡就钻入沼泽中，完全不见了踪影”）。
  - **对白占比**：极高，约60%-70%。情节推进、设定解说、人物塑造、笑点制造几乎全依赖对话完成。
  - **节奏**：叙述节奏快，战斗和危机场景切换迅速，环境描写和心理描写极简，重点放在“战前讨论”和“战后总结”上。

- **特色修辞 / 高频词 / 作者口味**：
  - **网络口语化**：大量使用吐槽式内心独白（“好吧，古月被萌到了！”、“为什么我会忽然想起艺术就是爆炸这句话”）。
  - **游戏化表达**：直接用“属性面板”展示武器数据（“器名：葬神戟\n属性：神魔\n附加战斗力：300000～600000”），用“王之财宝”命名敌人招式。
  - **中二招式名**：偏爱为技能取华丽名称（“苍鹰临首”、“有凤来仪”、“辟天裂地”）。
  - **冷幽默插入**：在紧张战斗后立即插入搞笑对白或吐槽，形成“残酷-轻松”的反差节奏。

- **章末钩子的三种主力款式**：
  1. **威胁升级型**（占比最高）：新强敌登场或危机浮现，如“一股巨大的压力使得所有人心神猛地一颤，一个黑色的影子慢慢的落下”。
  2. **谜底浮现型**：发现神秘物品或新地图，如“植物上面结着一枚果子，红光正是由这枚果子散发”。
  3. **未完对白/战斗型**：在关键时刻戛然而止，如萧青与冥海对峙时“三相神立即斩向冥海！！”。

## 人物 & 价值观
- **主角核心动机 + 性格关键词**：
  - **核心动机**：保护人类势力（不灭城），消灭威胁（虫神、海族）。务实且目标明确，不被外界干扰（“我的目标仅仅是连河果，不管他是不是叛徒，我只要将他的人头带回希望城”）。
  - **性格关键词**：冷静务实、智力型领袖、护短、游戏人间。表面冷酷（惩罚队员），实则培养（将惩罚包装成训练）。

- **主角的金手指是什么 + 作者如何避免“开挂无趣”**：
  - **金手指**：H病毒武装（绝对吞噬、元素化）、本命武器（葬神戟）、运气值设定、能力零点重置后抽取新能力。
  - **避免无趣的手法**：
    1. **金手指分阶段解锁**：属性面板上大量“？？？？”保留成长空间。
    2. **用智力包装金手指**：胜利往往靠“信息差”和“规则破解”（借正义武器破幻象），而非单纯力量碾压。
    3. **制造代价或限制**：能力有零点重置的冷却期，进化需要打破身体平衡。

- **典型反派形象 + 反派的“真实威胁值”如何维持**：
  - **典型形象**：压倒性实力的强者（冥海）、狡猾的巨型怪物（毒泽蛇王）、被篡改记忆的牺牲品（连河果）。
  - **维持威胁值的方式**：
    1. **让配角先被碾压**：冥海登场先轻松击败姜海和张飞猛，再与主角对抗。
    2. **揭示敌人的战斗智慧**：毒泽蛇王连续三次阴古月，展现狡猾。
    3. **制造信息不对等**：读者知道连河果被利用，但他自己不知道，形成悲剧性威胁。

- **本书价值观锚点**：
  - **爽感**（核心）：实力暴涨、收服强者、身份揭晓带来的震撼。
  - **智力碾压**：主角靠“看穿本质”和“规则破解”获胜，而非蛮力。
  - **守护与培养**：主角对队友和后辈的护短、教导（收徒章龙曹吉）。
  - **冷幽默**：在末世残酷中插入吐槽和搞笑对白，形成独特调性。

## 核心设定（**供“禁止直接抄袭”清单用**）
- **世界观公理**：
  1. 末世由虫潮引发，存在变异生物、H病毒、新人类进化体系。
  2. 存在“游戏系统”般的设定：战斗力数值、属性面板、技能名称、联合技。
  3. 世界存在多个禁地（如迷幻沼泽、古老密林），各有独特规则和危险。
  4. 存在多个势力博弈（如高氧森林的三个势力、神道组织）。
  5. 幻想体可以寄生在生物体内形成武器（如本命武器）。

- **主角专属设定**：
  - **金手指来源**：H病毒武装，拥有绝对吞噬、元素化等能力。
  - **身份秘密**：真实身份是“古月大人”，在游戏内外都有极高地位。
  - **真实立场**：人类势力的守护者，但行事风格务实冷静，不盲目正义。
  - **特殊机制**：能力会“零点重置”，需重新抽取；运气值设定影响赌局胜负。

- **专属角色 / 组织 / 地名（前5个）**：
  1. **古月**：主角，H病毒武装拥有者。
  2. **盖娅**：主角的辅助AI，负责解说设定、提供情报。
  3. **不灭城**：人类最后的希望之城。
  4. **虫神**：贯穿长线的终极反派。
  5. **高氧森林的三个势力**：拥有修改记忆能力的敌对组织。

- **专属物品 / 技能 / 概念名词（前5个）**：
  1. **葬神戟**：主角本命武器，属性面板化展示。
  2. **H病毒武装**：主角核心金手指系统。
  3. **幻想体胚胎**：可寄生生物体内形成武器的特殊存在。
  4. **演化物**：拥有特殊能力的物品（如上善若水、真假之心）。
  5. **零点重置**：主角能力会周期性重置的机制。

## 可借鉴的操作手法
- **情节层**：
  1. **“惩罚即训练”的反转**：将主角对队友的严厉包装成暗中培养，在真相揭晓时形成情绪反差。
  2. **“信息错位”制造期待**：让读者先于主角知道某个角色的误解（如连河果以为主角是迎接他的手下），形成“等着看笑话”的期待感。
  3. **“赌局收人”模式**：通过一场智力或运气对决收服强敌，同时获得关键情报，一举两得。
  4. **“双线并行，危机升级”**：主线轻松探索时，副线揭示更大威胁，在交汇点引爆全面冲突。

- **风格层**：
  1. **“属性面板”展示法**：将武器、技能、能力用“面板化”形式直接呈现，满足读者的“数据控”需求。
  2. **“冷幽默插入”节奏控制**：在紧张战斗后立即插入吐槽或搞笑对白（如“艺术就是爆炸”），形成“残酷-轻松”的反差节奏。
  3. **“中二招式名”增强记忆点**：为技能取华丽且易记的名称，让战斗更有画面感和辨识度。
  4. **“对话即解说”**：所有设定和世界观信息全通过角色对话释放，避免旁白说明打断节奏。

- **人物层**：
  1. **“废柴逆袭”式角色切入**：新角色先以“弱小/错误认知”登场（如曹吉以为自己是野鸡），再通过极端事件（死亡）觉醒真实力量。
  2. **“标志性动作/台词”快速定性**：角色登场时立即用一个动作或一句台词完成性格烙印（如刘朗的“章鱼式奔跑”、姜海的“冷面刀圣”）。
  3. **“导师型主角”的塑造**：主角不只是战斗者，更是培养者，通过制造危机激发队友潜力，体现更高维度的强大。
  4. **“角色功能化分工”**：每个配角在情节中承担明确功能（一人负责解谜、一人负责侦察、一人负责救场），工具性强但效率高。
