林家耀用九十五万买下红磡芜湖街一栋三层旧楼，从租客变成业主，银行账户只剩一万二，还欠赵老四和恒生银行贷款。他在记事本上划掉“搞到第一桶金”，夹入恒生银行审计报告复印件。

他在旺角茶楼与赵老四见面。赵老四质疑他资金来源，林家耀解释用楼抵押向银行借了四十万。赵老四提醒他走得太快容易踩坑，林家耀用数字证明合作半年已让赵老四赚了一万七，表示欠款照还。

林家耀到天台铁皮屋看望阿威母子。阿威还两千元，林家耀只收一千，将另一千作为修天台防水和栏杆的工钱交给阿威。他告诉阿威，帮他是因为他值得帮。

林家耀在铜锣湾避风塘与老姜见面。老姜指出他一年内从偷渡上岸到买楼，走得太快，根基不稳，身边只有阿威和赵老四一个半人。林家耀表示方向不错就够用，目标是中环。

他在天台记事本上写下新一行：“从一个人，变成一群人。”
