# 第一章 · 九龙城寨的第一顿饭

六月的九龙城寨，地下水沟的味道和烧腊香混在一起，林家耀分辨不出哪种更重。

他站在东头村道口，旅行袋的尼龙带勒进右肩，汗衫黏在后背，汗干了又湿。头顶上方，密密麻麻的冷气机像蜂窝一样钉在外墙上，滴水砸在铁皮檐上，噼啪声连绵成片，像永远下不完的雨。

三十六个钟头没吃过热食。从罗湖过关时吞了半条面包，到九龙塘转车时那半条也吐了——不是晕车，是饿过头。胃里空了太久，反倒不再叫，只是每隔一阵就抽紧，像有只手在慢慢拧。

远房表叔的名字叫陈德昌。林家耀捏着那张皱巴巴的地址条，按门牌摸到东头村道一栋唐楼三楼。铁闸紧闭，邻居阿婆隔着铁栅栏打量他，操着潮州口音粤语：德昌啊？上月搬咗啦，听讲去咗澳门。

阿婆说完就关了门，连杯水都没给。

他下了楼，在骑楼底站了一阵。口袋里还有一百二十块港币，是全部身家。一碗云吞面五蚊，一床位一晚十五蚊——他能撑三天。三天之内必须找到落脚处和进账门路，否则连睡天桥的资格都没有，九龙城寨的天桥底也分地盘。

大排档的镬气从巷口涌出来。柱侯牛腩在砂煲里咕嘟冒泡，八角和陈皮的香味像钩子一样扯着他的胃。林家耀没走过去。

他先在骑楼底蹲下来，放下旅行袋，摸出烟盒。最后一根万宝路，抖出来，想了想，又塞回去。明天才买得起新的。

这个动作让他想起泉州老家的码头。装船卸货，扛一包水泥两分钱，干一天够买两包烟。那时候他觉得穷。现在才明白，能吃饱的穷，和快饿死的穷，是两码事。

他站起来。腿有点软，膝盖骨像没上油的齿轮。不能拖了，先弄碗面。

大排档的塑料布棚下摆了六张折叠桌，午后三点，不是饭点，只坐了两个人。炉灶前老板正颠锅，铁镬铲敲在生铁锅沿上，节奏快得像打拍子。油烟冲上天花板，把塑料布熏成焦黄色。

林家耀还没走到档口，先听见另一边的动静。

“威仔，你老母下个月医药费仲欠两千，你条契弟几时还？”

三个小混混把一个人堵在巷口墙角。说话的那个穿花衬衫，头发搽得油亮，嘴里叼着牙签，右手食指戳着对方胸口。另外两个站他身后，一个板寸头，一个瘦长脸，手都插在裤袋里，裤袋鼓着。

被堵的人缩着肩膀，背贴着冷气机滴水的墙，T恤湿了一大片。他声音带着哭腔：雄哥，我下礼拜一定还，我阿妈呢次化疗——

“下礼拜？”花衬衫把牙签吐在地上，“你上礼拜都系咁讲。我大佬唔收息啊？两千变两千六，月底交唔出，叫你老母出院自己行返屋企。”

板寸头笑了一声。瘦长脸没笑，只是来回踅了两步，眼睛盯着巷口——看见林家耀，目光停了两秒，然后移开。

一个背着旅行袋的外乡人，不值得多看一眼。

林家耀没停步，径直走向大排档。他选了最靠近炉灶的位置坐下，背对着巷口，把旅行袋塞在脚边。

老板回头扫他一眼：食乜？

“云吞面。”

“七蚊。”

林家耀掏出钱，数了七个硬币放在灶台上。老板收钱，转身下面。

碱水味从滚水里蒸上来，混着云吞皮的面粉香。他盯着老板的手——左手磕开鸡蛋，右手铁镬铲翻锅，蛋液落进油锅，滋啦一声炸成荷包蛋。这双手粗糙，指节粗大，左手虎口有一道旧烫伤，疤痕发白。

系统在他脑海里跳出一行字：

「档案查询：九龙城寨东头村道·大排档登记人——周炳财。1959年与同乡合资经营周记大排档。1976年合伙人卷款潜逃，独力偿还债务。合伙人姓名：陈德昌。」

林家耀眼皮跳了一下。

陈德昌。他的远房表叔。

同一行字继续浮现：「陈德昌，原籍福建泉州，1976年卷款港币四万八千元潜逃澳门，未归。周炳财此后七年未再与人合伙。」

四万八。林家耀在心里算了一遍——1976年的四万八，够在九龙买一间唐楼。周炳财替他还了这笔债，独自撑了七年。

他端起茶杯，抿了一口。茶是普洱，泡得浓，苦味在舌根化开。

巷口的动静越来越大。

花衬衫不耐烦了，一把揪住阿威的领口：两千六，今个月底，少一个崩都唔得。

阿威被推得后脑撞在墙上，闷响一声。他眼眶红了，但没哭出来，嘴唇哆嗦着：雄哥，我真系冇——

板寸头上去就是一巴掌。阿威偏过头，鼻血滴在水泥地上，很快被热气蒸干。

林家耀放下茶杯。他看见阿威的眼神——不是求饶，不是愤怒，是一种被逼到墙角还在拼命算账的慌张。这种眼神他见过。在泉州码头，有个工友被债主堵在仓库里，也是这副神情。第二天工友跳了海。

面端上桌。碱水面条盘在白瓷碗里，汤底清澈，云吞皮薄得透出虾肉的粉红色。筷子搁在碗沿上，竹筷，洗得发白。

他没动筷子。

他站起来，走到灶台前。老板正往锅里下第二份面，头也不抬：仲要乜？

“周生，”林家耀声音不高，“我识得陈德昌。”

铁镬铲停在半空。

油锅还在滋滋作响，蛋液在热油里从边缘开始凝固。周炳财的手没动，手背青筋暴起。他慢慢转过头，盯着林家耀，眼睛像两把钝刀。

“你系边个？”

“佢表侄。”

沉默了三秒。周炳财把铁镬铲重重敲在灶台上，金属撞击声压过了油锅的噪音。他低声骂了句：顶你个肺。

然后绕过灶台，走到巷口。

林家耀没跟过去。他坐回原位，拿起筷子，夹起一箸面。面条入口，碱水味在舌尖化开，汤底的虾籽鲜味跟着涌上来。久违的热食落进胃里，暖意从腹部往四肢扩散。

他慢慢嚼，耳朵听着巷口的动静。

周炳财的声音粗得像砂纸：阿雄，叫你大佬今晚嚟搵我。阿威条数，我同佢讲。

花衬衫愣了一下：财叔，呢条友欠我大佬——

“我叫你大佬嚟搵我。”周炳财把话重复一遍，每个字都像从牙缝里挤出来，“定系你而家就想同我讲数？”

花衬衫张了张嘴，看了两个同伴一眼。板寸头摇头，瘦长脸已经往后退了一步。花衬衫咬咬牙，指着阿威：算你好彩。

三人沿着巷子走了，脚步快得像有人在追。

周炳财回到灶台前，继续颠锅。他没看林家耀，也没看阿威，只是把荷包蛋翻了个面，蛋黄在热油里颤了两下。

林家耀吃完了面。

他把空碗推到一边，倒了杯茶，慢慢喝。阿威还蹲在巷口墙角，鼻血已经不流了，留下一道暗红色痕迹从鼻孔到下巴。他用手背擦了一下，没擦干净，反而抹开了。

老板又端了一碗面过来，搁在灶台边上：食啦。

阿威抬头，看看老板，又看看林家耀。他站起来，腿还在抖，走到灶台前，端起碗，蹲在路边。筷子拿得不好，夹面时面条滑回碗里，溅出的面汤落在水泥地上，几秒就被暑气蒸干。

他吃得很急，像怕有人会抢。

林家耀从旅行袋里摸出半包烟——不是刚才那包，是另一包开封过的红双喜。他抽出一根，叼在嘴里，划火柴。火柴头擦过砂纸，火焰跳了一下，点燃烟丝。

他吐出一口烟。

阿威已经吃完了面，连汤都喝干净。他把碗放在地上，站起来，走到林家耀面前。

他嘴唇动了动，憋出几个字：多谢大佬。

林家耀没看他。烟夹在指间，青烟在潮湿的空气里散不开，像一层薄纱停在半空。

“阿威，”他把烟灰弹在地上，“你几岁？”

“二十二。”

“做乜工？”

“茶餐厅水吧。”阿威顿了顿，“前个月俾人炒咗。”

“点解？”

阿威沉默了一阵。拳头攥紧又松开。

“大佬叫我帮佢带粉落铺头。我唔肯。”

林家耀这才转过来看他。阿威的个子不高，瘦，但肩膀骨架宽，手掌大，指节上有旧的烫伤——茶餐厅水吧长期冲奶茶的痕迹。眼眶还是红的，但眼神不像刚才那么慌了，多了点倔。

“今晚有地方瞓？”林家耀问。

阿威摇头。

“去边度？”

“医院。”阿威说，“我阿妈住联合医院，夜晚探病时间过咗，我喺走廊瞓。”

林家耀吸了口烟，把烟头摁灭在烟灰缸里。

“听朝六点，嚟呢度搵我。”

阿威愣了一下：大佬，你贵姓？

林家耀站起来，把旅行袋甩上肩。尼龙带重新勒进肩膀，这次换到左肩。

“先别叫我大佬。”

他往东头村道的另一头走。身后的冷气机还在滴水，砸在铁皮檐上，噼啪声像永远下不完的雨。大排档的镬气追着他的背，柱侯牛腩的香味和沟渠的腐臭搅在一起。

周炳财没再看他一眼。铁镬铲敲在锅沿上，节奏还是那么快，像什么都没发生过。

阿威蹲在路边，看着林家耀的背影消失在骑楼底的阴影里。他把空碗端回灶台，对周炳财鞠了个躬，然后往医院的方向走。

走了两步，回头看了一眼巷口。

花衬衫三人早就不见了。但墙角的血迹还在，暗红色，正在被冷气机滴下的水一点点冲淡。

林家耀拐过街角，停下来。

他从口袋里摸出那张地址条，陈德昌的名字已经被汗水洇糊了。他把纸条揉成一团，塞回口袋，然后打开系统面板。

情报值余额：100。

刚才那一条花了多少？界面跳出一行小字：

「本次查询消耗情报值：15。剩余：85。」

不便宜。一句话十五点，够换三碗云吞面。

但换来的不只是一句话。

周炳财欠陈德昌一笔旧债，恨了七年。这条信息本身不值钱——但“有人知道这件事”这个事实，值。周炳财刚才出手，不是做好人，是封口。他不希望有人再提起陈德昌这三个字。

林家耀要的就是这个反应。

他不打算再找陈德昌。一个卷款潜逃的人，找到了也不会收留穷亲戚。但周炳财这条线，可以用。欠债七年还在恨的人，比欠债七年已经忘了的人，更可靠。

因为恨也是一种惦记。

他把烟盒掏出来，这次摸出最后一根万宝路，叼在嘴里，没点。

明早六点，阿威会来。

如果不来，就算了。

如果来了——就是第一个能用的人。

九龙城寨的夜晚来得比外面早。高楼把天空切割成不规则的条块，夕阳只能从缝隙里漏进来，把铁皮屋顶染成铁锈色。大排档的灯亮了，黄色灯泡套在红色塑料罩里，光晕模糊。

林家耀找了一间时钟酒店，三十蚊一晚。房间在三楼，窗户对着另一栋楼的墙，伸手能够到。他把旅行袋塞在枕头底下，躺下来。

天花板有水渍，形状像一张地图。

他闭上眼睛。胃里终于有了东西，暖意还没散尽。但四肢的酸软开始浮现，像潮水一样从骨头缝里漫出来。三十六个钟头的饥饿不会因为一碗面就消失，身体在秋后算账。

他没动。呼吸慢慢变匀。

脑子里还在算账。

一百二十块，扣掉七蚊面钱，三十蚊房租，还剩八十三。明天早餐五蚊，午餐七蚊，晚餐七蚊——一天伙食十九蚊。能撑四天。

四天之内，必须找到钱。

或者找到能帮他找钱的人。

阿威算一个。周炳财算半个。

系统算底牌，但不能常用。十五点一次，余额八十五，还能查五次。五次用完，情报值怎么赚回来，他还没摸清楚。

明天的事，明天再算。

他翻了个身，把脸埋进枕头。枕头有股漂白水味，刺鼻，但比沟渠的腐臭好闻。

窗外，九龙城寨的夜晚开始喧闹。麻将声从隔壁楼传过来，收音机里放着许冠杰的《半斤八两》，有人在天台上骂老婆，有狗在巷子里狂吠。

林家耀睡着了。
